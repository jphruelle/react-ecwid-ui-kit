Normal checkbox :
```
<CheckBox name="one" />
```

Custom label:
```
<div style={{ paddingTop: 10, paddingBottom: 10 }}>
  <CheckBox name="one" labelOn="LOL" labelOff="KEK" />
</div>
```

Disabled checkbox:
```
<div style={{ paddingTop: 10, paddingBottom: 10 }}>
  <CheckBox name="one" disabled />
</div>
```

Checkboxes sizes:
```
<div style={{ paddingTop: 10, paddingBottom: 10 }}>
  <CheckBox name="one" size="big" />
  <CheckBox name="one" size="tiny" />
</div>
```
